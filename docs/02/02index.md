# 餅の作り方

## 材料
- もち米……3合
- 水……350cc
- 片栗粉……適量

## 作り方
1. もち米を研ぎ、30分ほど水につけます。

2. 炊飯器にもち米と水をセットし、通常炊きを選択します。

3. 炊きあがったら打ち水をしながら、すりこぎ棒でこねましょう。

4. もち米の粒感がなくなるまで餅をつきます。

5. トレイに片栗粉をまぶし、餅を伸ばしましょう。

6.お好みのサイズに丸めたら完成です。
